# 投资收益计算器

一个基于 Streamlit 的投资收益计算工具，包含上证指数监控和分析功能。

## 功能特点

- 投资收益计算
- 上证指数实时监控
- 上证指数跌幅统计
- 数据可视化展示

## 安装依赖

bash
pip install -r requirements.txt

## 使用方法

bash
streamlit run investment_app.py


## 开发环境

- Python 3.8+
- Streamlit 1.10+

2. 运行应用：1
```bash
streamlit run investment_app.py
```

## 使用说明

1. 在左侧边栏输入初始投资金额
2. 输入当前投资价值
3. 查看实时计算的收益率和收益金额
4. 观察仪表盘显示的收益状态
